Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Nessa frase temos um termo em destaque, usando a tag <strong> (semântica).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica).

Nessa frase temos um termo em ênfase usando tag <em> (semântica).

Texto marcado (marca texto)

Podemos criar também um texto marcado usando a tag <mark>. Esse marcados está com essa tag, que é do HTML, mas está com css dentro da própria tag, para modificar a cor dessa marcação. Sem a marcação do CSS, a tag deixa o texto marcado apenas em amarelo.

Texto Grande e Pequeno

Esrtamos criando um texto grande e um texto pequeno nesse parágrafo. A primeira, tag <big> está OBSOLETA e já CAIU! A tag <small>, contudo, não está! A ideia dessa tag <small> é que seja utilizada com copyright ou pequenos comentários. Nesses casos específicos, ela tem um significado semântico, mas se for apenas pelo tamanho do texto, é preferível utilizar o CSS!

Texto Deletado

Podemos marcar um texto como excluído, com a tag <del>, para idicar que ele deve ser lido mas não considerado.

Texto Inserido / Sublinhado

Podemos marcar um texto como inserido, com a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado depois (como um espaço em branco para preenchimento posterior).

Existe também a tag <u>, que deixa o texto sublinhado, visualmente igual à tag <ins>, de inserido. No caso dessa tag, semanticamente, ela atende casos bem espefíficos, como o de erros de ecrita propositais ou para algusn nomes espefícicos chineses. No CSS, pode ser utilizado o "underline" para o mesmo fim visual.

Texto Sobrescrito e Subscrito

Para inserir, por exemplo, uma potência matemática, se utiliza a tag <sup>: y=x20+3.

Para inserir, por exemplo, uma fórula química, podemos utilizar a tag <sub>: H2O

Lista de Tags

No site da w3schools.com tem uma lista completa de tags. Nele, você consegue abrir cada uma, ver a descrição delas, a indicação de uso, exemplos de uso e, inclusive, dica de como alcançar o mesmo resultado visual com o CSS!
